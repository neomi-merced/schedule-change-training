# schedule-change-training# Schedule Change Training

An interactive training module for Nightingale College Registrar staff 
covering all four types of schedule changes — Add/Drop Period, Weeks 3–16, 
End of Semester, and Future Semester changes.

🔗 [View Training](https://neomi-merced.github.io/schedule-change-training/)

---

## About This Project

This module was built to train Associate Registrar staff on the policies, 
decision-making process, and activity workflows for every schedule change 
type they encounter throughout the academic year.

---

## What's Inside

- **4 Overview sections** — step-by-step process guides with annotated 
  screenshots and hover image popups
- **8 Scenarios** — branching decision-based practice cases
- **Flashcard study tool** — 8 key terms and policy rules
- **Final knowledge review** — 4 fill-in-the-blank questions covering 
  the full training
- **Completion screen** — quick reference table for all change types
- **Embedded feedback form** — Microsoft Forms for trainee responses

---

## Interaction Types

| Type | Where Used |
|---|---|
| Multiple choice with feedback | Add/Drop, Weeks 3–16, End of Semester, Future Semester scenarios |
| Drag & drop — match activity to recipient | Add/Drop Section 2 |
| Drag & drop — sort steps in order | End of Semester Section 1 |
| Fill-in-the-blank | Weeks 3–16 Section 2, Future Semester Section 2, Final Review |
| Flashcards | Study tool — accessible from Home |

## Instructional Design Notes

**ADDIE Framework** — Content was analyzed from existing PowerPoint training 
materials and a process guide document. Scenarios were designed around real 
workflow decision points that new staff commonly struggle with.

**Bloom's Taxonomy** — Questions progress from recall (flashcards, fill-in-the-blank) 
to application (scenario-based decision making) to analysis (sort steps, drag and drop).

**Learner feedback** — Interaction types were expanded beyond multiple choice 
based on direct feedback from trainees on a prior training module requesting 
more varied and visual interactions.
